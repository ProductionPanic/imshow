## Description
view images from the command line

## Usage
```
imshow <image>
```
the image should be a path to an image file (jpg or png)

## Installation
```
curl -s https://raw.githubusercontent.com/ProductionPanic/imshow/master/install.sh | bash
```

## Examples
```bash
imshow ~/Pictures/cat.png
```
